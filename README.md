# Azure Sentinel Lab

## Objectif
Déploiement d'un SIEM cloud-native sur Azure avec Microsoft Sentinel, règles KQL custom, connecteurs de logs et playbooks Logic App.

## Stack technique
- Microsoft Sentinel (SIEM)
- Log Analytics Workspace
- KQL (Kusto Query Language)
- Terraform (IaC)
- Logic Apps (SOAR / playbooks)
- Azure Arc (logs on-prem)
